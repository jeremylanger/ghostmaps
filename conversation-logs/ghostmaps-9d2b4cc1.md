## Human

how can i run two claude sessions at the same time with git worktrees?

---

## Assistant

Use the `--worktree` (`-w`) flag:

```bash
# Terminal 1
claude --worktree feature-auth

# Terminal 2
claude --worktree bugfix-123
```

This creates isolated directories at `.claude/worktrees/<name>/` with separate branches (`worktree-<name>`), so changes don't collide.

**Cleanup** is automatic — if no changes were made, the worktree is removed on exit. If commits exist, Claude prompts you to keep or remove it.

**Tips:**
- Name sessions for easy resuming: `claude --worktree feature-x -n "Add payments"`
- Each worktree needs its own `npm install`
- Resume later with `claude --resume "Add payments"`
